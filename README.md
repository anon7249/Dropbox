# Dropbox
A full-stack file management system inspired by Dropbox.

Users can:
Register & Login (JWT authentication)

Upload files

View their files

Download files securely (owner-only)

Generate public shareable links

Download files via public share link (no login required)


# This project demonstrates:

Token-based authentication

Secure file ownership checks

Public share-token architecture

RESTful API design


# Tech Stack
Backend

Python

FastAPI

PostgreSQL

JWT Authentication

Async database queries

UUID-based share tokens

Frontend

React

Axios

Blob-based file downloads

Clipboard API for share links
